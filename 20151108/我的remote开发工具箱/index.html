我的remote开发工具箱

在家里开发一段时间了，其中用到的相关工具如下：


  • 私有项目版本控制：taobao code

  • 个人项目版本控制：gitHub

  • 任务管理：teambition


  • 数据库设计：powerdesigner

  • 代码生成: codeSmith, 动软代码生成器


  • 开发工具：vs2015, sublime2

  • 正则工具：regexBuddy

  • json查看：jsonView


  • 即时沟通：QQ

  • 上传工具：FileZilla

  • 文件查找：everything

  • 截屏工具：picPick
